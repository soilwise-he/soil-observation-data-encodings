DESCRIPTION
-----------
Harmonized soil observation dataset from the ASAJA case study on olive
integrated production systems in Spain. Data has been standardized for
interoperability according to FAIR data principles for the SoilWise
European catalogue.

- **Geodata** — `Longitude_EPSG4326` and `Latitude_EPSG4326`, CRS in readme.txt
- **Sample Information** — `Upper_Depth_cm` and `Lower_Depth_cm`, UoM in readme.txt
- **Multiple IDs** — `Sample_ID` is the concatenation of `Reference` and `Plot`
- **Observed properties with Units of Measure (UoM)** — described in readme.txt
- **Procedures** — provided in readme.txt
- **Phenomenon time** — readme.txt claims that `Soil_Sampling_Date` is provided, but not in CSV
