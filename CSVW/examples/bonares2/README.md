# An experiment on using existing bonares (meta)data for soilwise interoperability

Harmonize a Bonares dataset with 2 tables using Soilwise harmonization guidance/tooling.

The [Long-term field experiment V140 Muencheberg from (launched in 1963) - Laboratory data (soil)](https://maps.bonares.de/mapapps/resources/apps/bonares/index.html?lang=en&mid=562631db-640b-4f13-8a08-2c2813d91be7) dataset has been downloaded from bonares platform under a CC-BY license (attribution: [Dietmar Barkusky](https://orcid.org/0000-0001-5241-8060), Leibniz Centre for Agricultural Landscape Research (ZALF))
 
- the metadata with table schema is available as *.bonares.json
- the cswv as -metadata.json
